Ionic
+++++

cli
===

::

    # add platform
    ionic platform ios 
    ionic platform android

    # Testing browser
    ionic serve [options]

    # build
    ionic build <platform>

    # run
    ionic run [--livereload|-l]

    ionic serve

Browser on Device
=================

::

    ionic browser add crosswalk    # adds 12.41.296.5
